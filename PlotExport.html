PlotDigitizer V2 — How to Use

▸ QUICK START (Auto-Trace)
  1. Open Image (Ctrl+O) or drag & drop
  2. Calibrate X: Pick X₁ + X₂, enter values, Apply
     (check Logarithmic for log axes)
  3. Calibrate Y: same process
  4. Switch to Auto-Trace mode (🖊)
  5. Click "Pick from Image" → click on the curve line
  6. Draw roughly over the curve (thick highlight pen)
  7. Click ⚡ Trace! → algorithm finds the curve
  8. Click ✓ Accept → points added to dataset
  9. Export CSV

▸ AUTO-TRACE TIPS
  • Pick color: Click directly on the curve LINE
  • Pen width: Wider = more forgiving, but may pick
    up nearby gridlines. Start with 30px.
  • Color tolerance: Higher = more matches. If you get
    noise, lower it. If curve is missed, raise it.
  • For log-log SOA plots: grid lines are usually gray,
    curve is colored → good color contrast!
  • Multiple curves: Add a new Dataset, pick new color,
    draw over next curve.

▸ MANUAL DIGITIZE
  • Switch to Digitize mode, click to add points
  • Right-click to delete nearest point
  • Ctrl+Z to undo

▸ NAVIGATION
  • Scroll wheel: zoom
  • Middle-click drag: pan
  • Drag & drop images directly onto canvas
